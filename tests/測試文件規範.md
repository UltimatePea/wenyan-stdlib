# 測試文件規範

**Author: Whisky, PR Worker**

## 測試文件命名
- 基本測試：`測試[庫名].wy` (例如：`測試字符串經.wy`)
- 單元測試：`單元測試[庫名].wy`
- 性能測試：`性能測試[庫名].wy`

## 測試目錄結構
```
tests/
├── 字符串經/
│   ├── 測試字符串經.wy
│   ├── 單元測試字符串經.wy
│   └── 性能測試字符串經.wy
├── 列經/
│   ├── 測試列經.wy
│   └── 單元測試列經.wy
...
```

## 測試要求
- 所有公共函數必須有測試
- 測試應涵蓋邊界條件
- 使用清晰的中文測試描述
