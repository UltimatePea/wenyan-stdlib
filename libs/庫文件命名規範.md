# 庫文件命名規範

**Author: Whisky, PR Worker**

## 庫文件命名
- 主要庫文件：`[庫名]經.wy` (例如：`字符串經.wy`)
- 工具文件：`[庫名]器.wy` (例如：`字符串器.wy`)
- 常數文件：`[庫名]常.wy` (例如：`算常.wy`)

## 目錄結構
```
libs/
├── 字符串經/
│   ├── 字符串經.wy
│   └── 字符串器.wy
├── 列經/
│   ├── 列經.wy
│   └── 列器.wy
├── 算經/
│   ├── 算經.wy
│   └── 算常.wy
...
```

## 命名慣例
- 使用古典中文
- 文件名直接對應功能
- 保持一致性和可讀性
